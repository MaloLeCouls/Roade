# Roade — Guide de démarrage

**Roade traite vos fichiers Excel et CSV directement sur votre ordinateur.**
Rien n'est envoyé sur internet, il n'y a aucun compte à créer, aucun mot de passe.

---

### 1. Installer (une seule fois)
Décompressez le dossier **`Roade`** où vous voulez (Bureau, Documents…).
C'est tout : il n'y a **rien à installer**, ni Python, ni quoi que ce soit.

### 2. Lancer Roade
Ouvrez le dossier `Roade` et **double-cliquez `Roade.exe`**.

- Une petite **fenêtre noire** s'ouvre : c'est **normal**, c'est le moteur de
  Roade. **Laissez-la ouverte** pendant que vous travaillez.
- Roade s'ouvre alors **tout seul dans votre navigateur** habituel.

### 3. La toute première fois : l'alerte Windows
Windows peut afficher **« Windows a protégé votre PC »**. C'est normal (le
programme n'est pas signé par un éditeur commercial) :
cliquez sur **« Informations complémentaires »**, puis sur **« Exécuter quand
même »**. À faire **une seule fois**.

### 4. Pour quitter
**Fermez la fenêtre noire.** (Fermer l'onglet du navigateur ne suffit pas : le
moteur continue de tourner.)

### 5. Où sont mes données ? Comment les sauvegarder ?
Tout votre travail est enregistré dans le dossier :

> **`Documents\Roade-projets`**

Pour faire une **sauvegarde**, il suffit de **copier ce dossier** (sur une clé
USB, un disque partagé…). Pour **mettre à jour** Roade, remplacez le dossier
`Roade` par le nouveau : **vos données ne sont pas touchées.**

---

### ⚠️ La règle d'or
**Ne sortez jamais `Roade.exe` de son dossier.** Il a besoin des fichiers qui
l'entourent pour fonctionner. Pour un accès rapide, créez plutôt un **raccourci**
sur le Bureau : clic droit sur `Roade.exe` → *Envoyer vers* → *Bureau (créer un
raccourci)*.

### Si rien ne s'ouvre
1. Patientez quelques secondes (le premier démarrage peut être un peu long).
2. Vérifiez que la **fenêtre noire est bien restée ouverte**.
3. Sinon, fermez tout et relancez `Roade.exe`.
